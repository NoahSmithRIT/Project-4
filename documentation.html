Project 4 Documentation

What technology and techniques I used to create the site:

I originally planned on using Materialize for page customization and easier access to layout options. The tutorial that I decided on using for the tic-tac-toe boards also used the row and col classes though, so they ended up being incompatible. Therefore, I switched to using the holy frail flexbox layout for my project. All of the coding for html, css, and js was done in Visual Studio Code. Version Control of the project repo was also maintained on VS Code. The images of the tic-tac-toe boards on the landing page were made in Inkscape. Lastly, the site is displayed using Github Pages.

What I had to learn:

I had to learn more about Javascript being used to manipulate html code to make the functionality of the second page, as well as to be able to better understand the Javascript code for running the tic-tac-toe games. To a lesser extent, I also had to learn to use the animate.css library that I used for the animations on the second page.

"Above and Beyond":

The buttons on the second page manipulate the html in a way that we never did in class by adding and removing classes. If I had the time to get to it, making the AI for the third page would have been my largest above and beyond. The tic-tac-toe board is mostly like it is from the tutorial. I changed the visuals of the board and changed some of the code so that you do not have to enter names for both players before starting a game. There are default names so that you only need to hit Play to start.
